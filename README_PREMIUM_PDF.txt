## 🎯 HOW TO USE (3 STEPS)

### Step 1: Wire Up Frontend Button
In your index.html or app.js, add a button:
```html
<button id="export-premium-pdf">📄 Export Premium PDF</button>
```

Then add the handler (from FRONTEND_HOOK_EXAMPLE.js):
```javascript
document.getElementById('export-premium-pdf').addEventListener('click', () => {
  exportPremiumPDF(yourBlueprintData);
});
```

### Step 2: Call the API
POST to `/api/export/premium-report-pdf` with your blueprint:
```json
{
  "blueprint": {
    "name": "TejBaat AI",
    "tagline": "AI-powered startup building",
    "marketScore": 85,
    ...
  }
}
```

### Step 3: User Gets PDF
Beautiful 8-page investor-ready report downloads to their device!

## 🔑 API ENDPOINT REFERENCE

REQUEST:
POST /api/export/premium-report-pdf
Content-Type: application/json

{
  "blueprint": {
    "name": "Your Startup",
    "tagline": "Your tagline",
    "marketScore": 85,
    ... (see PREMIUM_PDF_GUIDE.md for all fields)
  }
}

RESPONSE:
- Success: PDF file (application/pdf)
- Error: { error: "message" }

Status Codes:
- 200: PDF generated successfully
- 400: Invalid blueprint payload
- 500: PDF generation failed
