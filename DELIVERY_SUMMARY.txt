🔧 TROUBLESHOOTING:

Issue: "PDF generation failed"
→ Check: npm list puppeteer (should be installed)
→ Fix: npm install puppeteer --legacy-peer-deps

Issue: "Template not found"
→ Check: /public/startup-report.html exists
→ Verify: File path in server.js matches exactly

Issue: "PDF looks blank"
→ Check: Placeholder data being passed correctly
→ Verify: {{startupName}}, {{marketScore}}, etc. populated

💡 CUSTOMIZATION TIPS:

  Edit: font-family: 'Poppins'... → your font
